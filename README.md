# Email Dash

- O projeto propõe implementar um sistema web utilizando Python 3 e Flask, para que o usuário entre com sua conta Google e possa visualizar os seus contatos.
- Foi utilizado a biblioteca Person API da Google para coletar as informações dos contatos do usuário.
- O sistema está hospedado em um App Engine do Google Cloud, disponível em http://email-dash-python-flask-260823.appspot.com/
